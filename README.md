# LibraryManagementSystem
Семестр 4. Основи веброграмування та розробки мобільних застосунків. Лабораторна робота 1. Консольний застосунок Java.
## Бібліотека
+ Класи:
  + Бібліотека
  + Книга
  + Читач
+ Функціональності:
  + Додати читача
  + Додати книгу
  + Отримати інформацію про бібліотеку
  + Отримати інформацію про читачів
  + Отримати інформацію про конкретного читача
  + Отримати інформацію про книги
  + Отримати інформацію про конкретну книгу
  + Змінити інформацію про читача
  + Змінити інформацію про книгу
  + Видалити читача
  + Видалити книгу
  + Експортувати бібліотеку
  + Імпортувати бібліотеку
